# To-Do Swipe App

Минималистичный to-do лист в стиле iOS с жестами свайпа.

## Функции

- ✅ **Свайп влево** — удалить задачу
- ➡️ **Свайп вправо** — меню действий (Завершить / Завтра / Напомнить)
- 📅 Автоматический перенос просроченных задач
- 🌙 Тёмная тема iOS

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на Vercel

1. Запуш в GitHub
2. Импортируй на vercel.com
3. Deploy!

## Технологии

React 18 • Vite • Tailwind CSS • Lucide Icons
